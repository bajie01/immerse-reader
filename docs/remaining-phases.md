# 剩余开发计划

## Phase 1：阅读体验打磨（预计 2-3 周）

**目标**：从"能读"到"好读"，解决中文排版细节和阅读沉浸感。

### P1.1 排版深度打磨

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 标点挤压优化 | 中文引号、书名号、破折号的间距调整 | P0 |
| 两端对齐 | 中文正文 text-align: justify + 断词优化 | P0 |
| 首行缩进 | 正文段落首行缩进 2 字符（可选开关） | P0 |
| 西方文字嵌入 | 中英混排时英文自动换用适合的拉丁字体 | P1 |
| 行高优化 | 中文字体行高独立调节（1.8-2.0） | P0 |

### P1.2 阅读交互增强

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 阅读进度条 | 顶部精确进度指示 + 章节标记 | P0 |
| 目录侧栏 | 基于 h1-h4 自动生成，可点击跳转 | P0 |
| 阅读时长 | 根据字数 / 过往阅读速度动态计算 | P1 |
| 滚动记忆 | 退出阅读模式后恢复原文滚动位置 | P1 |
| 快捷键扩展 | j/k 滚动、t 回顶部、b 回底部 | P2 |
| 双页模式 | 宽屏下左右分栏（CSS columns） | P2 |

### P1.3 主题与自定义

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 自定义主题色 | 用户自由选择背景/文字/强调色 | P1 |
| 边距控制 | 左右边距独立调节 | P1 |
| 行高控制 | Popup 增加行高滑块 | P1 |
| 阅读专注模式 | 只高亮当前段，其余淡出 | P2 |

### P1.4 内容提取增强

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 图片自适应 | 大图点击放大、图注（figcaption）保留 | P0 |
| 代码块增强 | 行号、语言标签、一键复制 | P1 |
| 脚注悬浮 | 页面内脚注 hover 弹出内容 | P2 |
| 数学公式 | KaTeX / MathJax 渲染（学术场景） | P2 |

## Phase 2：AI 核心（预计 4-5 周）

**目标**：接入多模型 AI 能力，实现摘要、重点提炼、内容问答。

### P2.0 Model Router 架构

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 统一模型抽象层 | LLMService 接口：chat / summarize / ask | P0 |
| DeepSeek 接入 | 默认模型，OpenAI 兼容 API，国内直连 | P0 |
| Qwen 接入 | 通义千问 DashScope，备选模型 | P0 |
| GLM 接入 | 智谱 AI，备选模型 | P1 |
| OpenAI 兼容层 | 为用户自备 API Key 用户提供接入 | P0 |
| Anthropic 接入 | Claude 模型支持 | P1 |
| Base URL 自定义 | 支持代理 / 中转 / 本地 ollama 等 | P0 |
| 模型 fallback | DeepSeek 超时 → 自动切 Qwen / GLM | P1 |

### P2.1 摘要生成

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 一键摘要 | 页面级 → 200 字核心要点 | P0 |
| 分段摘要 | 每段/每节 → 一句话摘要 | P0 |
| 摘要长度控制 | 简短 / 标准 / 详细 三档 | P1 |
| 关键词/概念提取 | 自动识别文中的关键实体术语 | P1 |
| 摘要缓存 | IndexedDB 缓存已摘要页面 | P0 |

### P2.2 重点提炼

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 关键句自动高亮 | AI 识别文中最有信息量的句子，自动上色 | P0 |
| 提炼侧栏 | 侧栏展示提炼结果（分点列表） | P0 |
| 提炼导出 | 将提炼结果复制到剪贴板 | P1 |
| 自定义提炼粒度 | 用户可调整提炼的详细程度 | P1 |

### P2.3 内容问答

| 功能 | 说明 | 优先级 |

| 引用溯源 | 每条回答附原文引用段落 | P0 |

### P2.4 AI 配置与用量管理

| 功能 | 说明 | 优先级 |

## 推荐执行顺序

**Phase 1 优先做（按顺序）：**

1. SPA 路由检测（popstate + pushState 拦截）— 直接影响 SPA 站内导航可用性
2. 中文排版打磨（标点挤压 + 首行缩进 + 两端对齐）— 从"能看"到"好看"
3. 进度条 + 目录侧栏 — 长文阅读的掌控感
4. 图片自适应 + 图注保留
5. 阅读时长 + 滚动记忆
6. 自定义主题 / 边距 / 行高
